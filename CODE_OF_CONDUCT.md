# Code of Conduct

## Our pledge

We want `flutter_architect` to be a welcoming place for contributors of all backgrounds and experience levels.

## Our standards

**Do:**

- Be respectful and constructive
- Assume good intent
- Focus on what’s best for users of the package
- Give and accept feedback gracefully

**Don’t:**

- Harass, insult, or discriminate
- Troll or derail discussions
- Publish others’ private information
- Submit spam or malicious contributions

## Enforcement

Unacceptable behavior may result in warnings or exclusion from the project (issues/PRs closed, blocked from contributing).

Report concerns by contacting the maintainer via GitHub (@huzaifamirza0) or by opening a private security/contact channel if available.
